# Multi-Assistant AI Workspace

A **Streamlit web application** integrating multiple AI assistants using **LangChain** and **OpenAI LLMs**. This platform allows users to interact with AI for summarization, question answering, code assistance, and learning with quizzes — all in one interface.

---

## 🔹 Features

### 1. Summarizer
- Accepts **text input**, **file uploads** (`PDF`, `TXT`, `DOCX`), or **YouTube URLs**.
- Returns **concise summaries** using LLMs.

### 2. Chatbot (RAG)
- Retrieval-Augmented Generation chatbot.
- Can optionally use uploaded documents as a knowledge base.
- Provides **context-aware answers**.

### 3. Code Assistant
- Supports **code generation**, **debugging**, **optimization**, and **explanation**.
- Accepts code snippets and programming language as input.

### 4. Learning Assistant
- Offers **teach**, **quiz**, and **evaluate** modes.
- Can generate **interactive MCQ quizzes**.
- Provides **feedback** and **areas for improvement**.
- Accepts topics and optional document uploads for richer context.
